4.	程序功能描述
•	管理员可以添加删除员工，可以在后台查看实时数据；
•	管理员可以进行订单管理，套餐管理以及菜品管理、分类管理；
•	商户可以进行菜品分类和价格制定等
•	客户多平台点单； 
•	用户可以收藏自己感兴趣的菜 
•	用户可以查看收藏的菜
•	用户可以在菜品界面对该菜品进行评论             
•	用户可以查看自己的浏览历史
